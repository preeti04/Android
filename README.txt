Instruction

1) Download the Andriod SDK and its dependent components from the link 
https://developer.android.com/sdk/index.html

2) Install the SDK

3) Depends on system requirement if necessary download the enumlator for  arm x64 to support installation of the app

3) Append the System Variable PATH with these information
 PATH= ~./android/sdk/tools;android/sdk/platform-tools

4) Install andriod app

5) Run android.bat

6) Check troubleShoot.txt in case error occur
